IMPORT FB | Giải Thuật

TRY

  • Đọc file import .IMP với ngày Import Date được chọn
  • Lưu những gì đọc được vào bảng tạm Spend_FB_Trial
  • Đọc ngược từ bảng Spend_FB_Trial và hiển thị lên view
  • Cập nhật trạng thái ngài import được chọn là TRY

BOOK

  • Lưu danh sánh Spend FB hiển thị trên manh hình vào bảng Spend_FB
    Danh sách này có được:
    • TRY trước đó
    • REOPEN lại
  • Cập nhật trạng thái ngài import được chọn là BOOK

REOPEN

  1. Chọn ngày cần Reopen
  2. Chuyển trạng thái ngày được chọn sang REOPEN
  3. Xóa dữ liệu bảng Spend_FB thỏa: ngày Book_Date = Import_Date được chọn
  4. Đọc lại dữ liệu trong bảng Spend_FB_Trial thỏa: Book_Datre = Import_Date được chọn. Hiển thị lên view
